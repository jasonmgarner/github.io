Latitude vs. Weather

The purpose of this project was to analyze the changes that happen to the weather as you get closer to the equator. Weather information was obtained from OpenWeatherMap API and a dataset was compiled with over 600 cities.

Upon constructing the dataset, Matplotlib was used to utilized to plot different charcteristics of the weather and the relationhip to latitude. Characteristics observed included: temperature, humidity, cloudiness, and windspeed. This site provides visualizations to represent the analysis found and display the potential relationships between the weather and the proximity to the equator.
